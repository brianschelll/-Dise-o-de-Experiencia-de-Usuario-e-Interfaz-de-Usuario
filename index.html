Proyectos Personales

Desarrollé un sistema de registro de productos con control de stock, visualización de estadísticas, y notificaciones automáticas de reposición. Este proyecto incluyó el diseño de interfaces intuitivas y funciones robustas para el análisis de inventarios.

El sistema registra datos de clientes, productos vendidos, y genera informes detallados para analizar el rendimiento de ventas.Además, implementé un sistema de autenticación de usuarios y un panel de administración fácil de usar.
